# 🌐 Personal Portfolio Website

This is my **personal portfolio website** built to showcase my skills, projects, and experience as a **Full Stack Web Developer**.

This project was created as **Task 1 for my Full Stack Web Development Internship at Future Interns**.

## 🚀 Features

* Responsive design (Mobile & Desktop)
* Animated gradient background
* Skills and project showcase
* Contact form with email integration
* Resume download option

## 🛠 Tech Stack

* React.js
* HTML
* CSS
* JavaScript
* Vite
* EmailJS

## 📌 Sections

* Home
* About
* Skills
* Projects
* Contact

## ▶️ Run Locally

```bash
npm install
npm run dev
```
